# Download project in localhost
### Requirements
- React 17
- Firebase
- Node js
- JDK 11 (Open JDK)

### Steps to run software on localhost
- Download code
- Go to the root of project
- Run `npm install`
- Run `npm start`


Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
